    C 和 C++ 的区别与联系

    C++几乎是 C 的超集，只有很少的 C 的特性在 C++中不支持

    C++增加了需要方便实现面向对象特性的语法和封装，当然这些用 C 应该也能实现， 只不过 C++使其实现起来更简单。

    • C++ 改进了一些 C 中的缺点，比如 new 自动计算大小避免出错
    • C++ 增加一些语法糖，比如迭代器等
    • C++ 原生支持一些方便的库文件，比如 STL 库
